\doxysubsection{Detailed Description}
A non-\/owning string class (similar to the forthcoming std\+::string\+\_\+view) Note that, because a \doxyref{String\+Ref}{p.}{class_catch_1_1_string_ref} may be a substring of another string, it may not be null terminated.
